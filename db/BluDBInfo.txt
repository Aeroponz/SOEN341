[instructions]
To set up the DB you'll need to do the following in XAMPP.
	1. start Apache.
	2. start MySQL.
	3. click 'Shell' to open the XAMPP Terminal.
	4. type # cd mysql/bin
	5. type # mysql.exe -u root
	6. determine the root filepath for db/BluUserSetup.sql
	7. type >source [filepath]/db/BluUserSetup.sql;
	8. database "bluData" has been created along with three tables with dummy data.
